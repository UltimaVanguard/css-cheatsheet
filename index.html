CSS Snippet Cheatsheet

Select any of the content below and it'll automatically select the code to copy

Flexbox Row

Use these properties to create a Flexbox row layout.

.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

Flexbox Column

Use these properties to create a Flexbox column layout.

.column {
    display: flex;
    flex-direction: column;
}

CSS Grid Layout

Use these properties to create a 12 column layout using CSS grid

.css-grid {
    display: grid;
    width: 100%;
    grid-template-columns: repeat(12, 1fr);
}

Linear Gradients

This will create a background gradient from top to bottom

.card-pre{
    background-image: linear-gradient(
    rgba(232, 102, 236, 0.3) 0%,
    rgba(232, 102, 236, 0.6) 100%
    );
}

Box Transition Glow

Use transition and box shadow to glow on hover

.card,
.card .card-header {
    border-radius: 20px;
    transition: all 0.5s ease-in-out;
}
                    
.card:hover,
.card:hover .card-header {
    color: #000;
    box-shadow: inset 0px 0px 8px rgba(236, 19, 19, .7), 
    0px 0px 15px rgba(236, 19, 19, 1)
}

Overlay Card with Title

Use position properties and negative margins to raise elements higher than their natural starting points

.card-header {
    position: relative;
    margin-top: -20px;
}
